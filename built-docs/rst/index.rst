Plugin Index
------------

These are the plugins in the ans2dev.general collection:


Modules
~~~~~~~

* :ansplugin:`charts module <ans2dev.general.charts#module>` -- Generate high-quality charts using Plotly and save them as images.
* :ansplugin:`exa_facts module <ans2dev.general.exa_facts#module>` -- Gathers facts about Oracle Exadata Machine and rack.
* :ansplugin:`open_xl module <ans2dev.general.open_xl#module>` -- Read and update Excel files using openpyxl
* :ansplugin:`sar_facts module <ans2dev.general.sar_facts#module>` -- Collect system activity report (SAR) data for system performance monitoring.
